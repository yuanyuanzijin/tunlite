# Code of Conduct

## Our pledge

We as members, contributors, and maintainers pledge to make participation in the
tunlite community a welcoming experience for everyone, and to act in ways that
keep the community open, friendly, diverse, and healthy.

## Our standard

This project adopts the **[Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)**
as its code of conduct. Please read the linked document for the full set of
expectations and examples of acceptable and unacceptable behavior.

## Reporting

If you experience or witness behavior that violates this code of conduct, please
report it privately to the maintainers through
[GitHub's confidential reporting](https://github.com/yuanyuanzijin/tunlite/security/advisories/new).
All reports will be reviewed and handled discreetly.

## Enforcement

Maintainers are responsible for clarifying and enforcing this code of conduct and
may take fair, appropriate corrective action in response to any behavior they
consider inappropriate, following the Contributor Covenant's enforcement
guidelines.
